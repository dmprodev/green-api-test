# GREEN-API Тестовая страница

Тестовое задание — «Специалист технической поддержки 2 линии».

HTML-страница для работы с API GREEN-API. Реализованы 4 метода:

- **getSettings** — настройки инстанса
- **getStateInstance** — состояние инстанса
- **sendMessage** — отправка текстового сообщения
- **sendFileByUrl** — отправка файла по ссылке

## Запуск

Можно просто открыть `index.html` в браузере, серверная часть не нужна.

Опубликованная версия: https://green-api-test-orpin.vercel.app/
